# Carpe Diem | Shop Management System

A high-fidelity, premium Shop Management UI built with Next.js 16, Tailwind CSS, and Framer Motion. Designed for precision, efficiency, and a seamless user experience in industrial fabrication environments.

## 🚀 Features

- **Advanced Job Scheduling**: Pixel-perfect modal forms for scheduling cutting, polishing, and CNC operations.
- **Dynamic Shop Dashboard**: Real-time overview of fabrication jobs with status tracking and interactive filters.
- **Comprehensive Job Details**: Deep-dive into specific job parameters including stone types, edge profiles, and associated documentation.
- **Zod-Powered Validation**: Robust form handling with `react-hook-form` and Zod for reliable data entry.
- **Premium UI Components**: Custom-built, reusable components (Inputs, Selects, Modals, Toasts) following strict design tokens.
- **Global Notification System**: Custom toast system with smooth animations and high-visibility status updates.
- **Responsive Layout**: Sidebar-driven navigation with a centered, capped content area (1440px) for optimal readability.

## 🛠 Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **State Management**: Zustand
- **Forms & Validation**: React Hook Form + Zod
- **Animations**: Framer Motion
- **Icons**: Custom SVG Icon Library + Lucide React
- **Typography**: Proxima Nova & Inter

## 📦 Getting Started

### Prerequisites

- Node.js 18+ 
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/LordYemight/CarpeDiem.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎨 Design System

The project adheres to a curated design system defined in `src/app/globals.css`, featuring:
- **Primary Colors**: `#7A9705` (CarpeDiemGreen), `#5A6E1F` (Olive), `#667F01` (Moss)
- **Neutral Palette**: Specific grays and off-whites for premium depth (`#FAFAFA`, `#EEEEEE`, `#7C8689`)
- **Spacing**: Consistent 4px/8px based grid system.
